# Vinyard — Personal Portfolio Website

Vinyard is my personal portfolio website built to showcase my skills, projects, and journey as a developer. It serves as a central place where recruiters, collaborators, and fellow developers can learn more about me and explore the work I’ve done.

This project focuses on clean design, simple structure, and a professional online presence.

---

## 🌐 Live Purpose

The goal of this portfolio is to:

- Present my technical skills clearly
- Highlight selected projects and work
- Share a bit about my background and interests
- Provide an easy way to connect with me

A portfolio should reflect both personality and professionalism — this project is built with that mindset.
